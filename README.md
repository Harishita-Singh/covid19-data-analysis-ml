A complete data analysis and machine learning project on COVID-19 dataset using Python.  
This project includes data cleaning, exploratory data analysis (EDA), feature engineering, visualization, and predictive modeling.

---

## 📊 Dataset
- Source: GitHub Raw CSV  
- URL: https://raw.githubusercontent.com/SR1608/Datasets/main/covid-data.csv  

---

## ✨ Features

### 🔹 Data Analysis
- Data cleaning & preprocessing  
- Handling missing values & duplicates  
- Statistical summary  

### 🔹 Exploratory Data Analysis (EDA)
- Distribution plots  
- Scatter plots  
- Heatmap correlation analysis  
- Box plots for outliers  

### 🔹 Feature Engineering
- Created `death_rate` feature  
- Extracted month & year from date  
- Aggregated data using groupby  

### 🔹 Machine Learning
- Linear Regression model  
- Train-test split  
- Model evaluation (R² score, MSE)  
- Actual vs Predicted visualization  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

---

## 📈 ML Model Used
- Linear Regression  
- (Optional Upgrade: Random Forest Regressor)
